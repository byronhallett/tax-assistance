# Tax Assistance

A serverless function for zeit-now to fulfill tax assistance action on google

based on tax rates for 2019-2020:
| income             | percent_tax |
| ------------------ | ----------- |
| $0 – $18,200       | 0%          |
| $18,201 – $37,000  | 19%         |
| $37,001 – $90,000  | 32.5%       |
| $90,001 – $180,000 | 37%         |
| $180,000 and over  | 45%         |
